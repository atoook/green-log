# Requirements Document

## Project Description (Input)
観葉植物初心者は、植物の成長を写真で残したいが、現在の画像扱いは植物詳細体験として整理されていない。`plant-photo-schema-foundation` により、植物個体に複数の写真記録を紐づけ、代表写真を nullable に参照するための基盤仕様は存在するが、画像アップロード、画像ファイル保存・削除、写真 CRUD API、成長記録ギャラリー UI は対象外になっている。

この仕様では、植物詳細画面から現在表示している植物に対して画像をアップロードできるようにし、その植物に紐づく画像だけを時系列ギャラリーとして表示する。ユーザーは現在の画像枚数と上限を把握し、ギャラリー内の画像から代表画像を設定でき、植物一覧ではその代表画像がサムネイルとして表示される。画像削除時は確認ダイアログを表示し、代表画像を削除した場合は代表画像が未設定に戻る。
